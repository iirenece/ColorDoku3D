# ColorDoku3D
Versiones para ejecutar en distintas plataformas de Colordoku3D

## Android
1. Descargar *Aplicación Android*
2. Descomprimir el archivo
3. Configurar en 'Ajustes' del teléfono la instalación de paquetes de fuentes externas
4. Ejecutar *ColorDoku3D.apk* e instalar el paquete en tu teléfono
5. Iniciar la aplicación y... ¡a jugar!

## Web
1. Descargar *Aplicación Web*
2. Descomprimir el archivo
3. Acceder a la carpeta *www* y ejecutar el archivo *index.html*
4. Se abrirá la aplicación en el navegador web y...¡a jugar!
